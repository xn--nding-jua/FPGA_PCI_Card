Overview of the individual files
==================================
DIYFPGA.cpl		ControlPanel Application
DIYFPGA.inf		Driver Information File
fpgaprop.dll		PropertySheetPage for DeviceManager (16-bit DLL)
FPGATAB.dll		PropertySheetExtension for Windows Explorer
Readme.txt		This Readme-file
TVicHW32.dll		DLL to communicate with TVicHW32-Driver
TVICHW32.sys		General purpose device driver for WindowsNT/2k/XP
TVICHW32.VXD		General purpose device driver for Windows9x
Uninstall.bat		Uninstalls the driver and all files
